Library Management System (Web Application) :

A Library Management System built using Python and Object-Oriented Programming (OOP) principles. This web-based application helps manage library operations such as book records, user management,borrow books and return books tracking in an efficient and organized way.

Features :
📖 Add, update, and delete book records
👤 User registration and management
🔄 Issue and return books
🔍 Search books by title
📊 Track available and borrowed books
🔐 Basic authentication system for users/admin
